项目 06: 数组方法实验室 🔬 ​

🎯 项目目标 ​

通过交互式演示，深入理解 JavaScript 数组的核心方法。

📚 核心知识点 ​

转换类方法（返回新数组） ​

方法作用返回值原数组是否改变
map()对每个元素执行函数新数组（长度不变）否
filter()筛选满足条件的元素新数组（长度可能变）否
flat()展平嵌套数组新数组否

归约类方法（返回单个值） ​

方法作用返回值
reduce()将数组归约为一个值任意类型

查找类方法（返回元素或布尔值） ​

方法作用返回值
find()查找第一个匹配的元素元素或 undefined
some()是否有元素满足条件true/false
every()是否所有元素都满足条件true/false
includes()是否包含某个值true/false

排序类方法 ​

方法作用注意
sort()排序默认按字符串排序！数字需传入比较函数

📖 代码解析 ​

map() 详解 ​

javascript
const numbers = [1, 2, 3, 4];
const doubled = numbers.map(n => n * 2);
// doubled = [2, 4, 6, 8]

// 完整参数: map((元素, 索引, 原数组) => ...)
const indexed = numbers.map((n, i) => `${i}: ${n}`);
// ['0: 1', '1: 2', '2: 3', '3: 4']

filter() 详解 ​

javascript
const nums = [1, 10, 5, 20, 3];
const big = nums.filter(n => n > 5);  // [10, 20]

// 对象数组过滤
students.filter(s => s.score >= 80);  // 高分学生

reduce() 详解 ​

javascript
// 求和
const sum = [1, 2, 3, 4].reduce((acc, cur) => acc + cur, 0);
// acc: 0→1→3→6→10, 返回 10

// 求平均值
const avg = [1, 2, 3, 4].reduce((acc, cur) => acc + cur, 0) / 4;  // 2.5

// 统计词频
const words = ['apple', 'banana', 'apple'];
const freq = words.reduce((acc, word) => {
    acc[word] = (acc[word] || 0) + 1;
    return acc;
}, {});
// { apple: 2, banana: 1 }

sort() 陷阱 ​

javascript
// 错误: 默认按字符串排序
[10, 2, 1, 20].sort();  // [1, 10, 2, 20] ❌

// 正确: 传入比较函数
[10, 2, 1, 20].sort((a, b) => a - b);  // [1, 2, 10, 20] ✅

// 降序
[10, 2, 1, 20].sort((a, b) => b - a);  // [20, 10, 2, 1]

🚀 运行方式 ​

双击 index.html 在浏览器中打开，点击各方法按钮查看结果。

⚠️ 常见错误 ​

错误原因解决方法
数字排序错误没传入比较函数用 sort((a,b) => a-b)
原数组被修改sort/splice 会修改原数组先用 [...arr] 创建副本
reduce 初始值错误忘了传初始值或类型不对明确初始值类型
filter 返回空数组条件太严格检查回调函数的条件

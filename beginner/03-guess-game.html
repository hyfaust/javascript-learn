项目 03: 猜数字游戏 (Guess Number Game) ​

项目目标 ​

创建一个有趣的猜数字游戏，系统随机生成 1-100 之间的数字，玩家通过不断猜测并根据提示（太大/太小）来逐步接近答案。通过这个项目，你将学习 JavaScript 中的随机数生成、条件判断、循环思维、数组操作和用户交互等核心概念。

核心知识点 ​

  1. Math 对象: Math.random(), Math.floor() 的使用
  2. 条件判断: if-else, switch 语句
  3. 数组操作: push(), some(), forEach() 方法
  4. DOM 操作: 获取元素、修改内容、创建元素、操作 CSS 类
  5. 事件处理: 按钮点击、键盘事件（回车键提交）
  6. 输入验证: 检查空值、非数字、范围、重复猜测
  7. 游戏状态管理: 使用变量跟踪游戏进度
  8. CSS 动画: @keyframes 定义动画效果

关键概念解释 ​

随机数生成 ​

javascript
// 生成 1-100 之间的随机整数
Math.floor(Math.random() * 100) + 1
// Math.random() 返回 0-1 之间的数（不包含1）
// 乘以 100 后变成 0-100（不包含100）
// Math.floor() 向下取整，得到 0-99
// 加 1 后得到 1-100

// 1. 检查是否为空

运行方式 ​

  1. 直接用浏览器打开 index.html 文件
  2. 可以使用 VS Code 的 Live Server 插件
  3. 支持鼠标点击和键盘回车两种操作方式

操作说明 ​

  • 在输入框中输入 1-100 之间的数字
  • 点击"猜！"按钮或按回车键提交
  • 根据提示（太大/太小）调整下一次的猜测
  • 猜中后点击"重新开始"按钮开始新游戏

常见错误 ​

1. 输入非数字字符 ​

  • 现象: 输入字母或符号
  • 处理: 代码会显示"请输入有效的数字！"并清空输入框

2. 输入超出范围 ​

  • 现象: 输入 0 或 101 等超出 1-100 范围的数字
  • 处理: 代码会显示"请输入 1-100 之间的数字！"

3. 重复猜测 ​

  • 现象: 输入之前已经猜过的数字
  • 处理: 代码会提示"你已经猜过 X 了，请换一个数字！"

4. 游戏结束后继续输入 ​

  • 现象: 猜中后还想继续猜
  • 处理: 输入框和按钮会被禁用，必须点击"重新开始"

调试技巧 ​

  1. 查看目标数字: 打开浏览器开发者工具 (F12)，在控制台中可以看到目标数字
  2. 查看变量状态: 在 handleGuess 函数中添加 console.log() 查看猜测过程
  3. 修改范围: 尝试修改 getRandomNumber(1, 100) 的参数来改变游戏难度
  4. 测试边界: 尝试输入 1、100、0、101 等边界值测试验证逻辑

Promise 与 Async/Await

深入理解 JavaScript 异步编程的核心概念

一、Promise 基础

1. Promise 链式调用

使用 .then() 串联多个异步操作

new Promise(resolve => {
    setTimeout(() => resolve("步骤1完成"), 1000);
})
.then(result => {
    console.log(result);
    return "步骤2完成";
})
.then(result => console.log(result));

2. Promise 转 Async/Await

用更简洁的语法处理 Promise

// Promise 语法
fetchData().then(data => console.log(data));

// 等价的 async/await 语法
async function load() {
    const data = await fetchData();
    console.log(data);
}

二、模拟网络请求

1. 模拟 Fetch 请求

使用 setTimeout 模拟异步网络请求

function mockFetch(url) {
    return new Promise(resolve => {
        setTimeout(() => {
            resolve({ data: "来自 " + url + " 的数据" });
        }, 1000);
    });
}

2. 错误处理：Try/Catch

优雅地处理异步请求中的错误

async function fetchData() {
    try {
        const response = await mockFetch("/api/data");
        console.log("成功:", response);
    } catch (error) {
        console.log("失败:", error.message);
    }
}

三、Promise.all 并行请求

Promise.all 同时发起多个请求

等待所有 Promise 完成后统一处理结果

const [users, posts, comments] = await Promise.all([
    fetchUsers(),
    fetchPosts(),
    fetchComments()
]);

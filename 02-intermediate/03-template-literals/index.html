ES6 模板字符串 (Template Literals)

学习字符串插值、多行字符串和标签模板

一、字符串插值

1. 基础字符串插值

使用 ${} 在字符串中嵌入变量

const name = "小明";
const age = 18;
const msg = `我叫${name}，今年${age}岁`;

2. 表达式求值

${} 内可以执行任意 JavaScript 表达式

const price = 99;
const quantity = 3;
const total = `总价: ¥${price * quantity}`;

3. 在模板中调用函数

${} 内可以直接调用函数

function toUpperCase(str) { return str.toUpperCase(); }
const result = `大写: ${toUpperCase("hello")}`;

二、多行字符串对比

旧方法：字符串拼接

使用 + 和 \n 拼接多行字符串

var oldWay = "第一行内容\n" +
    "第二行内容\n" +
    "第三行内容";

新方法：模板字符串

使用反引号直接书写多行内容

const newWay = `第一行内容
第二行内容
第三行内容`;

三、标签模板 (Tagged Templates)

1. 简单标签模板

自定义函数处理模板字符串的各个部分

function highlight(strings, ...values) {
    return strings.reduce((result, str, i) => {
        return result + (values[i] ? `${values[i]}` : "") + str;
    }, "");
}
const name = "JavaScript";
highlight`学习 ${name} 很有趣`;

2. 货币格式化标签

自动将数字格式化为货币形式

function currency(strings, ...values) {
    // 将数值格式化为人民币形式
    const formatted = values.map(v => 
        typeof v === "number" ? "¥" + v.toFixed(2) : v
    );
    return strings.reduce((r, s, i) => r + (formatted[i-1] || "") + s);
}
const price = 199.9;
currency`价格是 ${price}`;

➡️ 箭头函数 vs 普通函数

语法对比

普通函数

function add(a, b) {
    return a + b;
}

const greet = function(name) {
    return 'Hello, ' + name;
};

箭头函数

const add = (a, b) => {
    return a + b;
};

const greet = (name) => `Hello, ${name}`;

// 简写形式（单表达式）
const multiply = (a, b) => a * b;
const square = x => x * x;  // 单参数可省略括号

何时使用？

✅ 适合用箭头函数

  • 回调函数（map, filter, setTimeout）
  • 不需要 this 的工具函数
  • 需要保持外层 this 的场景

❌ 不能用箭头函数

  • 对象方法（需要 this 指向对象）
  • 构造函数（需要 new 调用）
  • DOM 事件处理器（需要 this 指向元素）
  • 需要 arguments 对象的函数

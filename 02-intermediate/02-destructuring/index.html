ES6 解构赋值 (Destructuring)

交互式学习数组、对象和函数参数的解构语法

一、数组解构

1. 基础数组解构

按位置将数组元素赋值给变量

const arr = ["苹果", "香蕉", "橙子"];
const [first, second, third] = arr;

2. 跳过元素

使用逗号跳过不需要的元素

const arr = [1, 2, 3, 4, 5];
const [first, , third, , fifth] = arr;

3. Rest 剩余元素

使用 ... 收集剩余的所有元素

const arr = ["HTML", "CSS", "JavaScript", "React", "Vue"];
const [first, second, ...rest] = arr;

4. 交换变量值

无需临时变量即可交换两个变量的值

let a = 10, b = 20;
[a, b] = [b, a];

二、对象解构

1. 基础对象解构

按属性名从对象中提取值

const person = { name: "张三", age: 25, city: "北京" };
const { name, age, city } = person;

2. 重命名变量

使用冒号语法将属性名赋值给不同的变量名

const book = { title: "JavaScript指南", author: "李四" };
const { title: bookTitle, author: bookAuthor } = book;

3. 设置默认值

当属性不存在时使用默认值

const user = { username: "admin" };
const { username, role = "用户", status = "在线" } = user;

4. 嵌套对象解构

从嵌套的对象结构中提取深层值

const student = {
    name: "王五",
    scores: { math: 95, english: 88 },
    contact: { email: "wangwu@example.com" }
};
const { name, scores: { math, english }, contact: { email } } = student;

三、函数参数解构

1. 对象参数解构

直接在函数参数中解构对象

function greet({ name, age, greeting = "你好" }) {
    return `${greeting}，我是${name}，今年${age}岁`;
}
greet({ name: "赵六", age: 30 });

2. 数组参数解构

直接在函数参数中解构数组

function getDistance([x1, y1], [x2, y2]) {
    const dx = x2 - x1;
    const dy = y2 - y1;
    return Math.sqrt(dx * dx + dy * dy);
}
getDistance([0, 0], [3, 4]);

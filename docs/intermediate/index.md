# 阶段二：进阶特性

本阶段介绍现代 JavaScript 的关键特性，帮助你写出更优雅、更高效的代码。

## 项目列表

| 项目 | 名称 | 核心知识点 | 难度 |
|------|------|-----------|------|
| [01](./01-arrow-functions) | 箭头函数 vs 普通函数 | this 绑定、语法对比 | ⭐⭐ |
| [02](./02-destructuring) | 解构赋值 | 数组/对象解构、默认值 | ⭐⭐ |
| [03](./03-template-literals) | 模板字符串 | 字符串插值、多行文本 | ⭐ |
| [04](./04-promises-async) | Promise 与 async/await | 异步编程、fetch API | ⭐⭐⭐ |
| [05](./05-es-modules) | ES Modules | import/export、模块化 | ⭐⭐ |
| [06](./06-optional-chaining) | 可选链与空值合并 | ?. 操作符、?? 操作符 | ⭐⭐ |

## 学习建议

1. 每个项目花 1-2 小时
2. 先看代码，再运行，理解后自己修改实验
3. 重点理解 **this 绑定**和**异步编程**（这两个是面试高频题）
4. 完成每个项目的挑战任务

::: tip 注意
项目 05（ES Modules）需要通过 HTTP 服务器运行，不能直接双击 `index.html` 打开。可以使用 VS Code 的 Live Server 插件，或运行 `npx serve .` 命令。
:::

# 阶段一：基础入门

本阶段包含 7 个项目，帮助你掌握 JavaScript 核心语法与 DOM 操作。

所有项目都是纯 HTML/CSS/JS，**双击 `index.html` 即可在浏览器中运行**，无需任何构建步骤。

## 项目列表

| 项目 | 名称 | 核心知识点 | 难度 |
|------|------|-----------|------|
| [01](./01-todo-list) | 交互式待办清单 | DOM 操作、事件监听、数组增删 | ⭐ |
| [02](./02-calculator) | 简易计算器 | 函数封装、输入验证、错误处理 | ⭐⭐ |
| [03](./03-guess-game) | 猜数字游戏 | Math.random()、循环、条件判断 | ⭐ |
| [04](./04-weather-simulator) | 天气信息模拟器 | 对象、JSON、数据渲染 | ⭐⭐ |
| [05](./05-digital-clock) | 动态时钟 | setInterval、时间格式化 | ⭐ |
| [06](./06-array-playground) | 数组方法实验室 | map/filter/reduce/sort | ⭐⭐ |
| [07](./07-form-validator) | 表单验证器 | 正则表达式、事件处理 | ⭐⭐ |

## 学习建议

1. **按顺序学习**：每个项目都建立在前面项目的基础上
2. **先运行，再阅读**：先看到效果，再回头理解代码
3. **动手修改**：尝试修改参数、注释代码，观察变化
4. **完成挑战**：每个项目末尾的挑战任务是巩固知识的关键

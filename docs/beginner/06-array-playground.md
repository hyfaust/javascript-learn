# 项目 06: 数组方法实验室 🔬

## 🎯 项目目标
通过交互式演示，深入理解 JavaScript 数组的核心方法。

## 📚 核心知识点

### 转换类方法（返回新数组）
| 方法 | 作用 | 返回值 | 原数组是否改变 |
|------|------|--------|--------------|
| `map()` | 对每个元素执行函数 | 新数组（长度不变） | 否 |
| `filter()` | 筛选满足条件的元素 | 新数组（长度可能变） | 否 |
| `flat()` | 展平嵌套数组 | 新数组 | 否 |

### 归约类方法（返回单个值）
| 方法 | 作用 | 返回值 |
|------|------|--------|
| `reduce()` | 将数组归约为一个值 | 任意类型 |

### 查找类方法（返回元素或布尔值）
| 方法 | 作用 | 返回值 |
|------|------|--------|
| `find()` | 查找第一个匹配的元素 | 元素或 undefined |
| `some()` | 是否有元素满足条件 | true/false |
| `every()` | 是否所有元素都满足条件 | true/false |
| `includes()` | 是否包含某个值 | true/false |

### 排序类方法
| 方法 | 作用 | 注意 |
|------|------|------|
| `sort()` | 排序 | 默认按字符串排序！数字需传入比较函数 |

## 📖 代码解析

### map() 详解
```javascript
const numbers = [1, 2, 3, 4];
const doubled = numbers.map(n => n * 2);
// doubled = [2, 4, 6, 8]

// 完整参数: map((元素, 索引, 原数组) => ...)
const indexed = numbers.map((n, i) => `${i}: ${n}`);
// ['0: 1', '1: 2', '2: 3', '3: 4']
```

### filter() 详解
```javascript
const nums = [1, 10, 5, 20, 3];
const big = nums.filter(n => n > 5);  // [10, 20]

// 对象数组过滤
students.filter(s => s.score >= 80);  // 高分学生
```

### reduce() 详解
```javascript
// 求和
const sum = [1, 2, 3, 4].reduce((acc, cur) => acc + cur, 0);
// acc: 0→1→3→6→10, 返回 10

// 求平均值
const avg = [1, 2, 3, 4].reduce((acc, cur) => acc + cur, 0) / 4;  // 2.5

// 统计词频
const words = ['apple', 'banana', 'apple'];
const freq = words.reduce((acc, word) => {
    acc[word] = (acc[word] || 0) + 1;
    return acc;
}, {});
// { apple: 2, banana: 1 }
```

### sort() 陷阱
```javascript
// 错误: 默认按字符串排序
[10, 2, 1, 20].sort();  // [1, 10, 2, 20] ❌

// 正确: 传入比较函数
[10, 2, 1, 20].sort((a, b) => a - b);  // [1, 2, 10, 20] ✅

// 降序
[10, 2, 1, 20].sort((a, b) => b - a);  // [20, 10, 2, 1]
```

## ⚠️ 常见错误

| 错误 | 原因 | 解决方法 |
|------|------|---------|
| 数字排序错误 | 没传入比较函数 | 用 `sort((a,b) => a-b)` |
| 原数组被修改 | sort/splice 会修改原数组 | 先用 `[...arr]` 创建副本 |
| reduce 初始值错误 | 忘了传初始值或类型不对 | 明确初始值类型 |
| filter 返回空数组 | 条件太严格 | 检查回调函数的条件 |

# 项目 02: 简易计算器 (Simple Calculator)

## 项目目标

创建一个功能完整的简易计算器，支持基本的四则运算（加减乘除），包含计算历史记录功能。通过这个项目，你将学习 JavaScript 中如何处理用户输入、事件监听、错误处理等核心概念。

## 核心知识点

1. **变量声明**: `const` 和 `let` 的区别
2. **函数声明**: 传统函数和箭头函数的写法
3. **事件监听**: `addEventListener` 的使用
4. **错误处理**: `try-catch` 语句
5. **数组操作**: `push()`, `forEach()`, `slice()` 等方法
6. **字符串操作**: 模板字符串、`slice()`, `charAt()`
7. **DOM 操作**: 获取元素、修改内容、创建元素
8. **输入验证**: 防止无效的数学表达式

### 关键概念解释

#### try-catch 错误处理
```javascript
try {
  // 可能会出错的代码
  const result = eval(currentInput);
} catch (error) {
  // 出错时执行的代码
  display.textContent = '表达式错误';
}
```
如果 `try` 块中的代码出错，程序不会崩溃，而是跳转到 `catch` 块。

#### 事件监听器
```javascript
button.addEventListener('click', () => {
  // 点击按钮时执行的代码
});
```
当用户点击按钮时，箭头函数会被执行。

1. 直接用浏览器打开 `index.html` 文件

### 键盘快捷键
- `0-9`: 输入数字
- `+ - * /`: 输入运算符
- `.`: 输入小数点
- `Enter`: 计算结果
- `Backspace`: 删除最后一个字符
- `Escape`: 清除所有输入

## 常见错误

### 1. 表达式错误
- **原因**: 输入了不合法的数学表达式，如 `2++3` 或 `2//3`
- **解决**: 代码会自动显示"表达式错误"，按 C 键重新开始

### 2. 除以零
- **原因**: 任何数除以 0 在数学中是未定义的
- **解决**: 代码会显示"错误"，因为结果是 `Infinity`

### 3. 连续小数点
- **原因**: 一个数字中只能有一个小数点
- **解决**: 代码会自动阻止添加第二个小数点

### 4. 以运算符开头
- **原因**: 表达式不能以运算符开头（负号除外）
- **解决**: 代码会自动忽略无效的开头运算符
